RAG-System: Railways AnnualReport + Train Info

About This RAG System

RAG-System: Railways AnnualReport + Train Info is a Retrieval-Augmented Generation (RAG) application designed to answer queries about Indian Railways using both structured and unstructured data sources.

  • Frameworks: Built using LlamaIndex and OpenAI LLMs for advanced question answering.
  • Sources: Utilizes a PDF file (Indian Railways Annual Report 2023-24) and a JSON dataset (train information) as knowledge bases.
  • Query Engines: Employs LlamaIndex query engines to process both natural language and structured queries.
  • Vector Store: Uses ChromaDB for efficient vector storage and retrieval of document embeddings.
  • API: Powered by FastAPI to serve responses to user queries.
  • Deployment: Built and deployed on Hugging Face Spaces for public access.

This system demonstrates how modern RAG architectures can combine LLMs, vector databases, and traditional APIs to deliver rich, context-aware answers from both documents and structured data.
